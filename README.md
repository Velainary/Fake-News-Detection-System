# Fake News Detection System

Machine Learning project that classifies news articles as Fake or Real.

## Technologies

Python  
Pandas  
Scikit-learn  
Flask

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## How to Run

Install dependencies

pip install -r requirements.txt

Train model

python src/train_model.py

Run prediction

python src/predict.py

Run web app

python app.py
